# from-context-to-launch
From natural language to launched websites through Agent collaboration.

AI Native web creation workflow for non-developers.

This repository contains:

- `easy_web_skill_v2.md`  
  A web design & frontend generation skill pack for AI Agents.

- `from_context_to_launched.docx`  
  A whitepaper documenting how a non-developer built and launched a personal website through natural language collaboration with AI Agents.

- `prototype/index_v4.html`  
  The website prototype generated through this workflow.

Core idea:

> Humans decide what to build.  
> Agents figure out how to build it.

Built with Claude × Human collaboration.
